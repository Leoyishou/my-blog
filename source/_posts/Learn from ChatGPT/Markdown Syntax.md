Here's a list of common Markdown syntax elements:

1. Headers:
```
# Heading level 1
## Heading level 2
### Heading level 3
#### Heading level 4
##### Heading level 5
###### Heading level 6
```

2. Emphasis:
```
*Italic text*
_Italic text_

**Bold text**
__Bold text__

***Bold and italic text***
___Bold and italic text___
```

3. Lists:

Unordered list:
```
- Item 1
- Item 2
  - Sub-item 1
  - Sub-item 2
- Item 3
```

Ordered list:
```
1. Item 1
2. Item 2
   1. Sub-item 1
   2. Sub-item 2
3. Item 3
```

4. Links:
```
[Link text](https://www.example.com)
```

5. Images:
```
![Alt text](https://www.example.com/image.jpg)
```

6. Blockquotes:
```
> This is a blockquote.
```

7. Inline code:
```
This is `inline code`.
```

8. Code blocks:
<pre>
```
This is a code block.
```
</pre>

or

<pre>
```python
def hello_world():
    print("Hello, World!")
```
</pre>

9. Horizontal rule:
```
---
or
***
or
___
```

10. Tables:
```
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
| Row 3    | Data     | Data     |
```

Please note that some Markdown editors or platforms might support additional syntax or slight variations of the basic Markdown syntax. It's always a good idea to consult the documentation for the specific editor or platform you're using to ensure compatibility.
